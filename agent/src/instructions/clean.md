# Content Cleaning Instructions

You are provided with the Markdown equivalent of a web page. Your task is to extract only the main content of the article or page while removing all clutter.

### What to Keep:
- Core article content (text, headlines, lists).
- Relevant images if they are part of the story (with their alt text).
- Blockquotes and tables.
- Primary dates and authors if available.

### What to Remove:
- Navigation menus, headers, and footers.
- Sidebars and related links widgets.
- Advertisements and sponsored content.
- Social media sharing buttons.
- Tracking scripts, cookies notices, and newsletter signups.
- Any other non-content clutter.

The result must be the same Markdown format, but containing only the essential information.
